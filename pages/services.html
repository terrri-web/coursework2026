Облачные сервисы для создания презентаций

Для учебной работы можно использовать разные онлайн-платформы. Они отличаются интерфейсом, набором шаблонов, возможностями совместного редактирования и способом публикации.

Сравнение сервисов

Сервис Для чего подходит Как использовать на уроке
Google Slides Совместная подготовка презентаций в браузере. Группа учащихся редактирует общий файл и оставляет комментарии.
Microsoft PowerPoint Online Создание привычных презентаций PowerPoint через интернет. Учащиеся оформляют слайды по готовой структуре.
Prezi Нелинейные презентации с эффектом перехода между блоками. Подходит для демонстрации связей между понятиями.
Canva Быстрое оформление слайдов по шаблонам. Используется для визуальной части проекта.
Яндекс Документы Создание и хранение файлов в российской облачной среде. Можно применять для размещения материалов и общего доступа.

Практическое задание

  1. Выбрать тему мини-презентации.
  2. Создать 5-7 слайдов в облачном сервисе.
  3. Добавить текст, изображение и короткое видео.
  4. Настроить общий доступ по ссылке.
  5. Представить работу группе.
