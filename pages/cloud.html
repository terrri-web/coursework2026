Основные понятия облачных технологий

Облачные технологии дают доступ к программам и файлам через интернет. Пользователь работает с материалами в браузере, а данные хранятся на удаленных серверах.

Облачный сервис

Ключевые понятия

Облако

Среда, где файлы и программы доступны через интернет.

Сервис

Онлайн-инструмент для выполнения конкретной задачи.

Совместный доступ

Возможность работать над одним материалом нескольким пользователям.

Преимущества

  • не требуется установка программы;
  • материалы доступны с разных устройств;
  • можно быстро отправить ссылку на работу;
  • изменения сохраняются автоматически.

Ограничения

  • нужен стабильный интернет;
  • часть функций может быть платной;
  • нужно соблюдать правила безопасности аккаунта;
  • не все сервисы одинаково доступны в разных странах.
